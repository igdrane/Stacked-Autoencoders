# Stacked-Autoencoders
Projet MATLAB de classification d’images de chiffres manuscrits avec autoencodeurs empilés et couche softmax. Le code étudie l’effet du fine-tuning, l’impact d’un deuxième autoencodeur et l’optimisation des neurones cachés pour un compromis entre performance (≈ 99 % d’accuracy) et complexité du réseau.
